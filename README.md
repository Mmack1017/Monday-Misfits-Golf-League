# Monday Misfits Playoff Central

## Upload to GitHub
1. Open your `Monday-Misfits-Golf-League` repository.
2. Click **uploading an existing file**.
3. Upload all five files from this package at the repository root.
4. Click **Commit changes**.
5. Go to **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select **main** and **/(root)**, then click **Save**.

GitHub will show the public web address after deployment.
